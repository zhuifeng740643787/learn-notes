# PHP
[官方文档：http://php.net/manual/zh/index.php](http://php.net/manual/zh/index.php "官网文档")

## 序言
PHP, 即“PHP: Hypertext Preprocessor”(超文本预处理器) <br>
  + 一种__开源通用解释性__的**脚本**语言 <br>
  + 适用于Web开发并可嵌入html中 <br>
  + 目标是快速编写动态生成的html页面 <br>
