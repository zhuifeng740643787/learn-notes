    + 输出结果说明
        * . 当测试成功时输出。
        * F 当测试方法运行过程中一个断言失败时输出。
        * E 当测试方法运行过程中产生一个错误时输出。
        * R 当测试被标记为有风险时输出
        * S 当测试被跳过时输出
        * I 当测试被标记为不完整或未实现时输出
